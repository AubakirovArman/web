Прототип интерфейса · рабочая консоль

Кликабельный прототип всех экранов. Переход — по карточкам или через левое меню. Акцентный цвет переключается кружками вверху справа.

Как смотреть

Левое меню — переход между разделами. Кружки вверху справа — смена акцента (синий / индиго / бирюза / фиолет / графит). Это макет: данные демонстрационные.
