# Карта покрытия нормативных правовых актов (НПА) с разбивкой по типу процедуры

## Легенда

- ✅ **Реализовано** — НПА заведён в `lib/data/seed.ts` и используется в правилах/проверках.
- ⚠️ **Заведён, но не используется** — НПА есть в справочнике, но на него пока не ссылаются проверки.
