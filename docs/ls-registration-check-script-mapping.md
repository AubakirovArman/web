# ЛС → регистрация: сопоставление документов и проверок

Эта матрица показывает, какие проверки есть в коде, какие документы они покрывают после сопоставления новых типов документов из БД, и что остается на Gemma/manual.

## Каталог проверок, которые есть в коде

| checkId | Название | Метод | Что делает |
|---|---|---|---|
| `required_fields_check` | Обязательные поля заявления | rule | Проверяет заполненность обязательных полей для типа объекта и процедуры. |
| `required_document_presence_check` | Комплектность обязательных документов | rule | Проверяет наличие обязательных документов с учетом альтернативных документов. |
| `file_format_check` | Формат файла | parser | Сравнивает расширение загруженного файла с допустимыми форматами типа документа. |
| `ocr_quality_check` | Статус OCR/извлечения | ocr | Фиксирует статус распознавания, частичного извлечения и ошибки парсинга. |
| `expected_extracted_fields_check` | Обязательные извлечённые поля документа | parser | Проверяет, что из файла удалось извлечь поля, ожидаемые для этого типа документа. |
| `npa_imported_requirement_check` | Требования НПА к документу | manual | Показывает требования, извлеченные из НПА и привязанные к типу документа. Требует экспертной или гибридной проверки. |
| `gmp_certificate_check` | GMP и производственная площадка | hybrid | Проверяет срок действия, производителя, адрес и покрытие лекарственной формы в GMP. |
| `cpp_certificate_check` | CPP / регистрация в стране-производителе | hybrid | Проверяет CPP, срок действия и связь со страной производства. |
| `document_expiry_check` | Срок действия документа | parser | Проверяет извлечённое поле validUntil и выявляет просроченные или скоро истекающие документы. |
| `core_field_consistency_check` | Согласованность ключевых полей | hybrid | Сравнивает торговое наименование, МНН, дозировку, форму, производителя между заявлением и документами. |
| `shelf_life_consistency_check` | Срок годности | hybrid | Проверяет согласованность срока годности между ОХЛП, инструкцией, НДК и стабильностью. |
| `storage_consistency_check` | Условия хранения | hybrid | Проверяет условия хранения между ОХЛП, инструкцией и макетом. |
| `translation_length_check` | Полнота перевода RU/KZ | parser | Ищет существенное расхождение объема русской и казахской версии. |
| `docx_format_check` | Оформление DOCX | parser | Проверяет шрифт, размер и цвет текста для DOCX документов. |
| `required_sections_check` | Обязательные разделы ОХЛП/инструкции | parser | Проверяет наличие ключевых разделов в ОХЛП и инструкции. |
| `black_triangle_check` | Дополнительный мониторинг | hybrid | Проверяет наличие отметки о дополнительном мониторинге безопасности. |
| `pharmacovigilance_contact_check` | Фармаконадзор и контактное лицо | hybrid | Проверяет мастер-файл/краткую характеристику системы фармаконадзора, ПУР и наличие уполномоченного контактного лица в РК. |
| `bioequivalence_report_check` | Отчет биоэквивалентности | hybrid | Проверяет отчет БЭ, дозировку, лекарственную форму, референтный препарат и вывод. |
| `bioequivalence_waiver_check` | Биовейвер | hybrid | Проверяет обоснование отсутствия исследования биоэквивалентности. |
| `module3_content_check` | Модуль 3 / качество | hybrid | Проверяет наличие спецификаций, валидации и данных стабильности в Модуле 3. |
| `sterility_validation_check` | Стерильность | hybrid | Проверяет наличие подтверждения стерильности для стерильных ЛС/МИ. |
| `ls_reregistration_consistency_check` | Перерегистрация ЛС | hybrid | Проверяет номер РУ, срок действия и пострегистрационные данные для ЛС. |
| `ls_variation_consistency_check` | Внесение изменений ЛС | hybrid | Проверяет описание, обоснование, сравнительную таблицу и проекты документов для изменений ЛС. |
| `mi_registration_consistency_check` | Регистрация МИ | hybrid | Проверяет ключевые документы, испытания, СМК и класс риска МИ. |
| `mi_variation_consistency_check` | Изменения МИ | hybrid | Проверяет описание изменений, обоснование и обновленные инструкции/маркировку МИ. |
| `undocumented_variation_check` | Недокументированное изменение | hybrid | Ищет изменение значений, которое не отражено в описании или сравнительной таблице. |
