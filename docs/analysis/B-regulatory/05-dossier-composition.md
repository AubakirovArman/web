# B5. Состав регистрационного досье (документы и условия)

## ЛС — ОТД/CTD (Решение 78, приложение № 1; «Перечень документов в модулях»)
ОТД = 5 модулей. Модуль 1 специфичен для государств-членов; модули 2–5 общие.

### Модуль 1. Административные сведения
- 1.1 Содержание
- 1.2 Общая документация: **заявление**, оплата, доверенность, **ОХЛП**, **инструкция (ИМП/ЛВ)**, **макеты упаковок/этикеток**, сведения об экспертах, **система фармаконадзора** (мастер-файл, RMP/ПУР), **CPP** (сертификат фарм. продукта), **GMP**-сертификат, патенты/товарный знак, декларации, перечень стран регистрации.

### Модуль 2. Резюме ОТД
2.1 Содержание · 2.2 Введение · 2.3 Общее резюме по качеству · 2.4 Обзор доклинических · 2.5 Обзор клинических · 2.6 Резюме доклинических · 2.7 Резюме клинических (фармакология/эффективность/безопасность).

### Модуль 3. Качество (самый объёмный)
3.1 Содержание · 3.2 Основные данные:
- **3.2.S — Активная фармацевтическая субстанция** (общие сведения, производство, характеристика, контроль, стандартные образцы, упаковка, стабильность)
- **3.2.P — Лекарственный препарат** (состав, разработка, производство, контроль вспомогательных, контроль препарата, стандартные образцы, упаковка, стабильность)
- 3.2.A Дополнения (оборудование, посторонние микроорганизмы, новые вспомогательные)
- 3.2.R Региональная информация
- 3.3 Литература

### Модуль 4. Отчёты о доклинических (неклинических) исследованиях
4.1 Содержание · 4.2 Отчёты (фармакология, фармакокинетика, токсикология) · 4.3 Литература.

### Модуль 5. Отчёты о клинических исследованиях
5.1 Содержание · 5.2 Перечень КИ · 5.3 Отчёты:
- **5.3.1 биодоступность / сравнительная биодоступность и биоэквивалентность** (критично для generic)
- 5.3.2–5.3.7 прочие клинические отчёты, пострегистрационный опыт, литература.

### Условность состава по типу препарата (разд. III спец. требований)
| Тип | Особенности |
|---|---|
| Оригинальный | полные модули 4 и 5 (собственные доклиника + клиника) |
| **Generic** | модули 4/5 сокращены; **обязателен отчёт по биоэквивалентности** (5.3.1) или обоснование waiver |
| Биоподобный | сравнительные исследования с референтным биопрепаратом |
| Известная АФС | **мастер-файл АФС** (отдельная процедура) |
| ATMP (высокотехнологичные) | раздел IV спец. требований |
| Травяные/гомеопатические | упрощённые модули по монографиям |

### По процедурам
- **Регистрация** — полный CTD (по типу препарата).
- **Перерегистрация** — **модули 1 и 2** (актуализированные) + переоценка «польза–риск» (B2, п.127).
- **Изменения** — документы по типу изменения (IA/IB/II) и затрагиваемым разделам (Приложение № 19).

## МИ — досье (Решение 46, Приложение № 4 + п.10/п.27)
Регистрационное досье = заявления (Прил. 2,3) + документы по перечню **Приложения № 4**, среди которых (драйвер — класс риска):
- доказательства соответствия **Общим требованиям** (Реш. 27)
- **протоколы технических испытаний** (Реш. 28)
- **отчёт по оценке биологического действия** + биобезопасность (Реш. 38)
- **отчёт о клиническом доказательстве** + клинические/клинико-лабораторные испытания (Реш. 29)
- **отчёт по анализу рисков**
- **отчёт о стабильности** (обоснование срока хранения)
- данные о **валидации ПО** (если есть ПО)
- данные о **стерилизации** (если стерильное)
- **эксплуатационный документ/инструкция**, **маркировка**, технические характеристики
- данные о **СМК** (Реш. 106), производственные площадки
- сведения о неблагоприятных событиях, план постпродажного мониторинга
- для измерительной функции — испытания для утверждения типа средств измерений
