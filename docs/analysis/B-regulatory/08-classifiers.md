# B8. Классификаторы и справочные стандарты

## ЛС — типы лекарственных препаратов (Решение 78)
Тип определяет состав досье и проверки (B5):
- **Оригинальный** (с новой АФС / известной АФС)
- **Воспроизведённый (generic)** — однокомпонентный/многокомпонентный
- **Биоподобный (биоаналог)**
- **Гибридный**
- **Биологический** / иммунобиологический (в т.ч. вакцины)
- **Растительный (травяной)**
- **Гомеопатический**
- **Радиофармацевтический**
- **Препарат из крови/плазмы**
- **Хорошо изученного применения (well-established use)**
- **Орфанный**
- **Высокотехнологичный (ATMP / advanced-therapy)**

> В MVP `param-product-type` = 13 значений (original/generic/hybrid/biological/biosimilar/vaccine/herbal/homeopathic/radiopharmaceutical/orphan/blood/well-established/advanced-therapy) — соответствует.

## ЛС — прочие справочники
- **АТХ-классификация** (код + наименование) — поле заявки.
- **МНН** (INN) — поле заявки (kz/ru/en).
- **Лекарственные формы**, способы введения, единицы дозировки — справочники полей.
- **Фармакопея ЕАЭС** (Решение № 100) — стандарты качества/методики.
- **Классификатор** (Решение № 65) — уточнить предмет.

## МИ — классы потенциального риска (Решение № 173)
**4 класса:** **1** (низкий) · **2а** · **2б** · **3** (высокий). Класс определяет:
- объём регистрационного досье,
- требования к **СМК** (Решение № 106),
- необходимость клинических испытаний и глубину доказательств,
- глубину экспертизы.

## МИ — прочие классификаторы/требования
- **Номенклатура МИ** (Решение № 177) — вид изделия.
- **Классификатор видов МИ** (Решение № 48), **областей применения** (Решение № 62), **перечень видов** (Решение № 42).
- **Общие требования безопасности и эффективности МИ** (Решение № 27) — базовый критерий соответствия + маркировка/эксплуатационная документация.
- **СМК** (Решение № 106) — система менеджмента качества по классу риска.
- Методики испытаний: технические (№28), биологическое действие (№38), клинические (№29).

## Надлежащие практики (GxP) — ЛС
- **GMP** — Решение № 77 (производство)
- **GCP** — Решение № 79 (клинические)
- **GLP** — Решение № 81 (лабораторные)
- (уточнить: №83/85/87/89 — фармаконадзор/биоэквивалентность/маркировка/дистрибуция)

## Связь с реализацией (для C)
- ЛС типы препаратов — покрыты полем `param-product-type`.
- **Классы риска МИ (1/2а/2б/3) — в MVP как драйвер НЕ используются** (нет правил документов МИ). Это ключевой недостающий классификатор для паритета МИ.
- Номенклатура/области применения МИ — отсутствуют как справочники в форме.
- АТХ/МНН/ЛФ — присутствуют как поля; источники-справочники (актуальность) не синхронизированы.
